Problem Solving

Do these steps for once in the repo of problem-solving directory
  • Extract folder PS for Problem-solving question
  • Copy folder and paste it inside the repo for problem-solving
  • Initialize a package.json by using npm init -y inside your repo directory
  • Install jest package globally by using the command => npm i jest -g

For daily solving & test solution
  • Run terminal inside PS folder in your repo directory
  • To test the solution,write => jest .//test.js
  • Make sure there is a package.json file in your main directory
